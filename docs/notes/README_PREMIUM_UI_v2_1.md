# Pulau Jawi v2.1 Premium Kids UI Upgrade

Upgrade ini fokus kepada kekemasan UI family-friendly dan pengalaman kanak-kanak 5–8 tahun.

## Fail utama yang dinaik taraf
- `index.html`
- `game-map.html`
- `lesson-game.html`
- `app.css`
- `js/pulau-jawi-effects.js`
- `js/pulau-jawi-data.js`

## Penambahbaikan utama
- Warna lebih ceria dan premium kids: candy pastel, mint, sunshine, sky, pink dan purple.
- Animasi tambahan: floating avatar, sparkle, sun pulse, tree sway, button shine, correct pop, wrong shake, reward confetti.
- Game page lebih hidup untuk wide screen smartphone.
- Audio Web Audio lebih ceria: blip, magic, win, victory, oops.
- Voice masih guna `speechSynthesis` sebagai demo sahaja; untuk produksi disarankan MP3 suara kanak-kanak Melayu sebenar.
- 10 pulau kekal untuk umur 5–8 tahun.
- Zafri/Zainab kekal sebagai avatar konsisten.

## Nota penting
Fail sensitif seperti Supabase, ToyyibPay, admin logic, dan database config tidak diubah.
